# kindling-test

A test repository for [kindling](https://github.com/redwoodjs/redwood), the agentic task orchestration system developed under the RedwoodJS project.

## Purpose

This repository serves as a sandbox for testing kindling workflows, task force execution, and dev container integration. It is not a production project.

## Dev Container

The repository uses a dev container with kindling tooling. The local kindling dev container configuration (`.devcontainer/kindling/`) is not tracked — it is generated or managed per environment.

## Status

Early stage. No application source is present yet.
