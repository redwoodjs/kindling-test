# kindling-test

A sandbox repository used to test the [Kindling](https://github.com/redwoodjs/kindling) task force agent system.

## Purpose

This repository serves as a test target for Kindling — a devcontainer-based orchestration system that coordinates AI agents across development tasks. It has no application code of its own; its value is as an environment where Kindling workflows can be exercised end-to-end.

## Usage

Kindling runs inside the devcontainer. To run tasks against this repository:

```bash
# From within the devcontainer
kindling launch -
```

Refer to the [Kindling documentation](https://github.com/redwoodjs/kindling) for full usage.
